# Aldo's CloudPRNT Server

This server receives online orders from Aldo's Pizzeria kiosk app  
and sends them to a Star Micronics CloudPRNT thermal printer.

## Endpoints

### `POST /submit`
Send the ticket text from your kiosk app.

### `GET /cloudprnt`
Printer fetches the ESC/POS job.
